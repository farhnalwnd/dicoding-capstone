# Weaknesses

Scraper block event loop. LinkedIn block bot easily. Fallback description generic hurt match. Use async scraper with proxy.
Vector search slow. Python loop over all jobs inefficient. Switch to MongoDB Atlas Vector Search or dedicated vector DB.
Skills list hardcode IT only. Non‑IT domains unsupported. Generate dynamic skill list or expand taxonomy.
Parser name extraction fragile. Regex fail varied CV formats. Scanned PDFs lack OCR. Apply NER and OCR.
MongoDB client global. Connection leak risk. Use FastAPI lifespan/dependency injection.
